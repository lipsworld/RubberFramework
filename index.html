O que é?


Rubber é um framework CSS simples e leve - é claro que ele contém algumas rotinas e bibliotecas em Javascript, mas seu manuseio e prototipação se dá apenas via HTML e CSS3. O Rubber Framework foi criado a partir do 1140px CSS Grid, de Andy Taylor.


O Framework original possuía layout responsivo simples, no qual o grid de 12 colunas do modelo padrão era "empilhado" em colunas simples para tablets e smartphones. O Rubber Framework adiciona novas classes CSS para uso do grid em tela inteira, não importando qual a largura trabalhada. Também é possível exibir colunas sem espaçamento, "coladas" umas às outras.


As novas classes .full-width e .tight irão permitir o uso da largura total da tela, sem extinguir os antigos .container e .row - ambos usados como anteriormente, em 1140px.

Com a nova classe .full-width, é mantido o sistema de "grid" em 12 colunas, porém em tela cheia, sem limitações de largura.


Como no sistema do 1140px CSS Grid, cada novo .container ou .full-width pode conter diversas subdivisões horizontais, separadas por DIVs com classes .row, que por sua vez podem conter qualquer arranjo de colunas que some um total de 12, acionadas por classes próprias (.onecol, .twocol, .threecol, .fourcol e assim por diante, lembrando sempre de adicionar a classe .last à última coluna de cada .row.
